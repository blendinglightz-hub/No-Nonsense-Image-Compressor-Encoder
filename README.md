# No-Nonsense-Image-Compressor-Encoder
Single-file, client-side image compression and format conversion. Nothing is uploaded. All processing happens in the browser.

**[Open Tool](https://blendinglightz-hub.github.io/No-Nonsense-Image-Compressor-Encoder/index.html)**

## Formats
Accepts: JPEG, PNG, WebP, GIF, BMP, AVIF, JXL
Outputs: JPEG, PNG, WebP, AVIF, JPEG XL

## Features
- Batch compress with global settings (quality, effort, lossless)
- Before/after file size and savings per image
- Skip individual files without removing them from the queue
- Download per-file or as a ZIP
- No install, no build step, no server
